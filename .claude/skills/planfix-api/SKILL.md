# /planfix-api — Управление задачами и контактами

## Алгоритм

1. Вызови `get_tasks` для получения списка задач
2. Вызови `create_task` для создания новой задачи
3. Вызови `get_contacts` для получения списка контактов

## Формат ответа

```
## Задачи Planfix

Найдено: 42 задачи

### Последние задачи
1. Подготовить отчёт — В работе — Иванов И.
2. ...
```

## Примеры

```
/planfix-api задачи
/planfix-api создать задачу "Подготовить отчёт"
/planfix-api контакты
```
